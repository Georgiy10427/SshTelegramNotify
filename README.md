# SshTelegramNotify
The bot notifies about successful SSH connections to the server.

<h1><b>Install</b></h1>
<h3> Debian/Ubuntu Server <br> </h3>
<code>
  sudo apt git
</code><br>
<code> 
git clone https://github.com/Georgiy10427/SshTelegramNotify.git 
</code><br>
<code>
cd SshTelegramNotify 
</code><br>
<code>
sudo chmod 777 install.sh
</code><br>
<code>
  ./install.sh
</code><br>
